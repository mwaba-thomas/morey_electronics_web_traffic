# Morey Electronics Web Traffic Analysis

[Morey Electronics]([https://moreyelectronics.com?utm_source=chatgpt.com](https://www.morey.shop/)) is an electronics company that makes it possible for customers in Zambia to buy electronics through small, flexible instalments starting from as little as **$0.60 (K10) per day**. This model helps people access devices needed to stay connected, work, learn, and grow. Whether someone is a bike rider, marketeer, farmer, student, or entrepreneur, the platform enables safer and more reliable access to technology, helping more people participate in the digital economy on their own terms.

This project contains a **SQL-based Exploratory Data Analysis (EDA)** of Morey Electronics website traffic data collected between **29 November and 5 December 2024**. The analysis focuses on understanding **purchase intent** and user behavior patterns on the platform. Since not all visitors were authenticated, the project treats **unique IP addresses as individual users** for unauthenticated traffic analysis.

## Data Analyzed

The project analyzed the following areas:

* User activity:

  * Authenticated vs unauthenticated users
  * Returning visitors
  * Average number of visitors per day
* Traffic trends:

  * Dates with the highest website traffic
* Geographic analysis:

  * User locations and regional distribution
* Basket activity:

  * Most common basket product categories
  * Basket interaction behavior
* Purchase intent:

  * Percentage of users who both interacted with baskets and returned to the site across multiple days

## Tools Used

* SQL for exploratory data analysis
* Power BI for data visualization
* CSV datasets

## Goal

The goal of this project is to demonstrate how implementing automated authentication methods such as **Google Sign-In** or similar social login systems could improve customer tracking and conversion performance.

The analysis suggests that better authentication systems could:

* Improve identification of returning users across devices and sessions
* Increase basket-to-purchase conversion rates
* Support automated remarketing campaigns such as:

  * Reminder emails for incomplete purchases
  * Personalized product recommendations
  * Retargeting campaigns for high-intent users
* Provide more accurate customer behavior analytics
* Improve long-term customer retention and engagement

Overall, this project highlights how combining web traffic analytics with user authentication strategies can help Morey Electronics better understand customer behavior and optimize the online purchasing journey.
